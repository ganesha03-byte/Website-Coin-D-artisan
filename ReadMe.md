# Coin d'Artisan

Coin d'Artisan is a handcrafted products e-commerce website developed as part of a web development project. The platform showcases artisanal products through a modern, responsive, and user-friendly interface.

## Features

- Responsive website design
- Product showcase and catalogue
- Product detail pages
- Shopping cart interface
- Customer reviews section
- Contact page
- Image galleries and multimedia content
- Clean and modern user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Web Design

## Pages

### Product Page
